### About
A demake of 'The Binding of Isaac', created as a project for the Object-Oriented Programming course (2023/24).

### Commands
- **Move**: `W` `A` `S` `D`
- **Attack**: `↑` `←` `↓` `→`
- **Resize**: `P` `M`
